**📦 ft_transcendence – 42HUB**

*This project has been created as part of the 42 curriculum by anony, ertrigna, kearmand and vdeliere.*

**🌐 About the Project**

**42Hub** is a social network designed for 42 student (and beyond), focused on interaction, communication, and collaboration.

**✨ Features**
- 🔐 Log in via the 42 Intranet (OAuth)
- 👤 Create and manage a public profile
- ➕ Follow / ❌ Unfollow users
- 💬 Real-time chat with other users
- 📁 Upload files and images
- 🚀 And many more features to enhance community interaction

**🔧 Installation**

To run this project:
- git clone <repository_url>
- cd ft_transcendence

Add the required .env files (see configuration if needed)

- make
- make dev

**🌐 Usage**

- https::/localhost:5173

**🧭 Supported Browsers**
- 🦊 Firefox
- 🌍 Chrome
